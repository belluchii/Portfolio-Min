Acerca de mí

Soy Desarrollador Full Stack con más de 2 años de experiencia construyendo aplicaciones web end-to-end. Me importa escribir código claro, tomar buenas decisiones técnicas y trabajar bien en equipos distribuidos.

Actualmente trabajo de forma remota para G2I / Outlier, evaluando y validando desafíos de programación, revisando implementaciones y analizando la calidad técnica de soluciones. También evalúo respuestas de modelos de lenguaje como ChatGPT y Claude: detecto errores, justifico correcciones y refino outputs con criterio técnico real.

Mi stack principal es JavaScript / TypeScript con React y Next.js en el frontend, y Node.js + Express / Fastify en el backend. Desarrollo APIs REST, implemento autenticación con JWT y trabajo con MongoDB y PostgreSQL. Acompaño el desarrollo con testing (Jest / React Testing Library), Docker y Git / GitHub.

Busco una posición remota en un equipo con buenas prácticas y problemas reales. Tengo inglés profesional y me interesa crecer en un entorno que exija tanto en lo técnico como en la comunicación.
